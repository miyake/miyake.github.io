問 1

問 2

\begin{align} 0 &= dU \\ &= \left( \frac{\partial U}{\partial T} \right)_V dT + \left( \frac{\partial U}{\partial V} \right)_T dV \\ &= C_{\mathrm{v,m}} dT + \left( T \left( \frac{\partial P}{\partial T} \right)_V - P \right) dV \ \ \ \ \ \ \ \ ( \because \text{ (2) } ) \\ &= C_{\mathrm{v,m}} dT + \left( T \cdot \frac{N_1R}{V-bN_1} - \left( \frac{N_1RT}{V-bN_1} - \frac{aN_1^2}{V^2} \right) \right) dV \ \ \ \ \ \ \ \ ( \because \text{ (1) } ) \\ &= C_{\mathrm{v,m}} dT + \frac{aN_1^2}{V^2} dV \end{align} から、 \begin{align} \Delta T &= - \frac{aN_1^2}{C_V} \int_{V_1}^{V_2} \frac{dV}{V^2} \\ &= \frac{aN_1^2}{C_V} \left[ \frac{1}{V} \right]_{V_1}^{V_2} \\ &= \frac{aN_1^2}{C_V} \left( \frac{1}{V_2} - \frac{1}{V_1} \right) \end{align} がわかる。

問 3

\begin{align} \left( \frac{\partial S}{\partial V} \right)_U = \frac{P}{T} \gt 0 \end{align} なので、不可逆過程である。
