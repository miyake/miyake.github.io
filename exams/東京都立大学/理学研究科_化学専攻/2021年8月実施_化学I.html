(イ)

理想気体の状態方程式より \begin{align} \frac{PV}{T} = (\gamma-1) C_V \tag{a} \end{align} が成り立ち、与えられた条件より \begin{align} PV^n = A \ \ \ \ ( \text{ 定数 } ) \tag{b} \end{align} が成り立つ。 温度が $T_1, T_2$ のときの体積を $V_1, V_2$ とし、圧力を $P_1, P_2$ とする。 \begin{align} \Delta w &= - \int_{V_1}^{V_2} AV^{-n} dV \ \ \ \ \ \ \ \ ( \because \text{(b)} ) \\ &= \frac{A}{n-1} \left( V_2^{-n+1} - V_1^{-n+1} \right) \\ &= \frac{1}{n-1} \left( P_2 V_2 - P_1 V_1 \right) \ \ \ \ \ \ \ \ ( \because \text{(b)} ) \\ &= \frac{(\gamma-1)C_V}{n-1} (T_2-T_1) \ \ \ \ \ \ \ \ ( \because \text{(a)} ) \end{align}

(ウ)

\begin{align} \Delta q &= - \Delta w + C_V (T_2-T_1) \\ &= \frac{n-\gamma}{n-1} C_V (T_2-T_1) \end{align}
