  • 環境によっては、数式の表示に時間がかかる場合があります。

積分定数を $C$ と書く。

(1)

\begin{align} \int \frac{4}{x^2 + 2x - 3} dx &= \int \left( - \frac{1}{x+3} + \frac{1}{x-1} \right) dx \\ &= - \log_e \left| x+3 \right| + \log_e \left| x-1 \right| + C \\ &= \log_e \left| \frac{x-1}{x+3} \right| + C \end{align}

(2)

\begin{align} \int \frac{e^x}{e^x + 2} dx &= \int \frac{\left( e^x + 2 \right)'}{e^x + 2} dx \\ &= \log_e \left( e^x + 2 \right) + C \end{align}

(3)

\begin{align} \int x e^{2x} dx &= \frac{1}{2} x e^{2x} - \frac{1}{2} \int e^{2x} dx \\ &= \frac{1}{2} x e^{2x} - \frac{1}{4} e^{2x} + C \end{align}

(4)

\begin{align} \int \frac{\sin x}{\cos x} dx &= - \int \frac{\left( \cos x \right)'}{\cos x} dx \\ &= - \log_e \left| \cos x \right| + C \end{align}
